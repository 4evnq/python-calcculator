# Консольний калькулятор статистичних показників

## Опис проєкту
Консольна програма, яка зчитує числові дані з текстового файлу, обчислює базові статистичні показники (середнє, медіану, мінімум та максимум) і зберігає результати в окремий файл

## Розподіл ролей
* **Студент А (Team Lead / Integrator)**: Відповідає за зчитування даних із файлу, інтеграцію модулів, консольний інтерфейс та збереження результатів.
* **Студент Б (Developer / Tester)**: Відповідає за реалізацію математичних статистичних функцій та перевірку даних на помилки.

## Інструкція із запуску
1. Створіть файл `input.txt` у папці `data/` та додайте туди числа (кожне з нового рядка).
2. Запустіть програму з кореневої директорії проєкту командою: 
   `python src/main.py`
3. Ознайомтеся з результатами у консолі або у створеному файлі `data/output.txt`.
